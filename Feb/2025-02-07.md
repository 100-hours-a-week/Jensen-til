#### Pandas의 apply와 map
- apply
    - 적용 대상 : 축(행/열)
    - 적용 인자 : 함수
- map
    - 적용 대상 : 시리즈 혹은 데이터프레임 전체
    - 적용 인자 : 함수 / 딕셔너리
